# Objetos de aprendizagem digitais

Entre 2008 e 2015 eu liderei uma equipe multidisciplinar de programadores e ilustradores, ocasião na qual construímos inúmeros objetos de aprendizagem digitais para Educação a Distância e lousas eletrônicas, tendo participado de projetos como [Aulas Interativas](https://irpagnossin.com/projects/aulas-interativas), convênio USP-Univesp para a criação do curso de [Licenciatura em Ciências da Univesp]((https://irpagnossin.com/projects/univesp)), Tidia-Ae e [RedeFor]((https://irpagnossin.com/projects/redefor)). Esses recursos educacionais foram criados usando Java Applets e Flash com Active Script 3, e vários deles se comunicavam com um _Learning Management System_ via SCORM 2004; tecnologias que atualmente estão ultrapassadas. Este repositório visa resgatar o código-fonte desse trabalho e, se viável, portá-los para tecnologias mais novas.
